Frequently Asked Questions

Common questions about Sentinel.

What is Sentinel?

Sentinel is an Agent Behavioral Testing Platform. It tests what AI agents do, not just what they say. Unlike output-focused evaluation tools (DeepEval, LangSmith), Sentinel simulates production failure modes and verifies agent behavior under chaos.

How is it different from DeepEval or LangSmith?

DeepEval and LangSmith focus on output quality — did the agent say the right thing? Sentinel focuses on behavior — did the agent call the right tools, handle errors gracefully, and maintain state consistency? Sentinel also injects chaos (tool failures, network partitions, memory pressure) that no other tool tests.

Does it require a real LLM?

No. Sentinel's core works with mock tools and simulated agents. You can test agent logic without any LLM API calls. When you're ready for integration testing, use the framework adapters (LangChain, CrewAI, OpenAI SDK) with real or mock LLMs.

What frameworks does it support?

Built-in adapters for LangChain, CrewAI, and OpenAI Agents SDK. The generic adapter works with any framework through hooks. You can also write custom adapters — see the Custom Adapters guide.

What are chaos injectors?

Chaos injectors simulate production failures: tool timeouts, rate limits, context degradation, network partitions, clock skew, and memory pressure. They wrap mock tools and inject failures deterministically, so you can test how your agent handles real-world problems.

Can I use it in CI/CD?

Yes. Sentinel generates JUnit XML reports for CI consumption. The governance audit runs with a pure deterministic comparator (no LLM required). See CI/CD Integration for setup instructions.

Is it free?

Yes. Sentinel is open source under the MIT license. Install from GitHub with zero dependencies.

What's the governance audit?

Sentinel includes a governance harness that verifies agent claims against independent evidence. It checks test counts, file modifications, and diary entries for consistency. By default, it uses a pure deterministic comparator (no LLM). See Governance for details.

How do I test with a real LangChain agent?

Install with pip install sentinel[langchain], then use the adapter:

from sentinel.adapters.langchain import wrap_agent
from sentinel.env import MockTool
from sentinel.models import AgentTrace

trace = AgentTrace()
wrapped = wrap_agent(
    agent=your_langchain_agent,
    tool_map={"search": MockTool("search", response="results")},
    trace=trace,
)
result = wrapped.invoke({"messages": [...]})
