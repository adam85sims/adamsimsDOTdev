Installation

Get Sentinel running in under a minute.

Requirements

  • Python 3.11 or later
  • pip or any PEP 517-compatible package manager

Install from GitHub

The recommended way to install Sentinel:

Minimal install (zero dependencies)

pip install git+https://github.com/adam85sims/Sentinel.git

With LangChain adapter

pip install "git+https://github.com/adam85sims/Sentinel.git[langchain]"

With all adapters

pip install "git+https://github.com/adam85sims/Sentinel.git[adapters]"

Development install

git clone https://github.com/adam85sims/Sentinel.git
cd Sentinel
pip install -e ".[dev]"

Optional Dependencies

Sentinel's core has zero dependencies. Install only what you need:

ExtraInstallProvides
langchainpip install sentinel[langchain]LangChain adapter
crewaipip install sentinel[crewai]CrewAI adapter
openaipip install sentinel[openai]OpenAI SDK adapter
governancepip install sentinel[governance]YAML config support
otelpip install sentinel[otel]OpenTelemetry export
devpip install sentinel[dev]pytest, ruff, coverage
allpip install sentinel[all]Everything above

Verify Installation

python -c "import sentinel; print(sentinel.__version__)"
# Should print: 0.1.0

Run Tests

cd Sentinel
pytest tests/ -v
✅ You're ready! Head to Your First Test to write your first behavioral test.
