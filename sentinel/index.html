Quick Example

from sentinel.env import MockTool, EnvironmentBuilder
from sentinel.models import AgentTrace
from sentinel.assertions import assert_tool_called, assert_no_tool_errors

# Create mock environment
env = EnvironmentBuilder().mock_tool("search", response={"results": []}).build()

# Record agent behavior
trace = AgentTrace()

# Run your agent (replace with actual agent code)
env.tools["search"](query="test")

# Assert behavioral properties
assert_tool_called(trace, "search")
assert_no_tool_errors(trace)

Why Sentinel?

88% of AI agents fail in production. The dominant failure modes are operational — tool errors, memory issues, edge cases — NOT hallucination. Yet no testing tool simulates these failures before deployment.

Sentinel fills that gap. It's the only testing platform that:

  • Mocks the entire environment — not just the LLM, but tools, APIs, and databases
  • Injects real chaos — network partitions, clock skew, memory pressure, cascading failures
  • Asserts behavior — tests what agents DO, not just what they SAY
  • Works with any framework — LangChain, CrewAI, OpenAI SDK, or custom
