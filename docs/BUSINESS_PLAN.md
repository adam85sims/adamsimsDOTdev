## 7. OPERATIONS & TOOLING

Everything runs on a single Fedora workstation with a local-first philosophy:

- **Inference:** Ollama, llama.cpp, Headroom routing proxy
- **Agents:** Hermes Agent, OpenCode, Claude Code (multi-provider strategy)
- **Creative:** ComfyUI on AMD ROCm (local image generation)
- **Comms:** Tailscale mesh, systemd services, self-hosted infrastructure
- **Version control:** GitHub (public for OSS, private for client work)
- **Documentation:** Markdown-first, always in-repo

**Why this matters to clients:** I don't depend on someone else's uptime to
build your thing. My dev environment is self-contained, backed up, and
reproducible.
